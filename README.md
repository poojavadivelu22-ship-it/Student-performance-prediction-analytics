# Student Performance Prediction - Data Analytics Project

## Demo
https://student-performance-prediction-analytics-fk3yqcvjc3k7hd6jee7fc.streamlit.app/

## 📊 Overview
This project analyzes student academic performance data to identify factors affecting student results and predict whether a student may pass or fail.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📁 Dataset
The dataset contains student details such as:
- Attendance percentage
- Study hours
- Assignment scores
- Internal marks
- Previous semester marks
- Final result (Pass/Fail)

## 📈 Analysis
- Student attendance analysis
- Pass vs Fail comparison
- Study hours analysis
- Internal marks distribution
- Performance prediction using machine learning

## 💡 Insights
- Students with high attendance score better marks
- More study hours improve academic performance
- Low assignment scores may lead to failure
- Internal marks strongly affect final results
